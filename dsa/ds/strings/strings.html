    String Length

    1. We know that the string is terminated by a null character ('\0').
    2. The string is first read from the index being initialised to zero
    3. The index is incremented after every read
    4. As soon as the index reaches the null character - the value of the index is the length of the characters
      
  # include<stdio.h>
  main(){
    char str[40];
    int i=0;
    printf("Enter the string: \n");
    gets(str);
    while(str[i]!='\0'){
      i++;
    }
    printf("\n the length of the string is: %d",i);
  }
