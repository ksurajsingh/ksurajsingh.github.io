Animations

Very similar to transitions, but can give you modular options.
tailoring specific needs to specific properties
use it with @keyframes <name> for your speicifcation.
you can specify stuff like :
  • from - to
  • percentage like - 10% 20%
animation has the following properties
Name
the name of the keyframe
Duration
duration of the animation
Timing-function
a function for time speed relevance
Delay
delay in time to start the animation
Iteration-count
The number of times to repeat an animation
direction
Direction of the animation ,
If you make a the animation linearly from left to right
default left-to-right
reverse right-to-left
alternate-reverse right-to-left then in the next iteration left-to-right. [ just alternate - does the same thing starting from the start [ initial point ]]
fillmode
controlling the state before the animation starts and after the animation ends

8.Keyframes

This is not an animation property but yeah, so we can actually make the animation go from
from to to or even make breakpoints to controls then at breaks using
percentages with relevance to time

For example if there are 5 percentages with 20% intervals per state
and the duration is 5 seconds we will get to see each transition in 1 second
box14 
    
@keyframes container3{
  0%{
    transform: translateX(0);
  }
  20%{
    transform: translateX(10vw);
  }
  40%{
    transform: translateX(20vw);
  }
  60%{
    transform: translateX(30vw);
  }
  80%{
    transform: translateX(40vw);
  }
  100%{
    transform: translateX(50vw);
  }
}
with steps for each state
They key frames will define states using %
and the step() will define the state for each of those keyframes
