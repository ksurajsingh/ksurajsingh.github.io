Table

  1. table tag is used to create a table.
  2. thead tag is used for the header section of the table
  3. th tag is used to create a header for the table
  4. tbody tag is used to create the body for the table
  5. tr tag is used to create a row in the table
  6. td corresponding to the cell this fills in the specified data in a cell that is td in that particular row specified by tr
  7. tfoot similar to thead
  8. secondary use of these tags is to follow semantics in order to apply appropriate CSS for a corresponding relations

  9. rowspan = n One particular row uses specified number of rows .
    Essentially one row takes the space of specified number of rows
  10. colspan = n Self-explainatory

  11. caption can be used to give captions to the elements inside whom these are specified



List

There are three main types of list
ordered unordered and Defination
  1. Ordered List have bullets like numbers, roman numbers, alphabets
  2. Unordered list have bullets that do not follow any orders like : dots, star, arrow pointers
  3. dl: is the defination list tag
    dt: defination title
    dd: defination defination/data


Use the type attribute to specifiy the type of bullets
example: type="i" , type="I", type="1"[default], type="a", type="A"
