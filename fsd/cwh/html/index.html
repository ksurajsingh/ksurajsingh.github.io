Structure of HTML

Self closing tags close by themself.
The other ones are called paired tags

  2. html lang="en" helps the parser to understand that : this particular html is written in "en" language
  3. title is used to give the title of the type that appears in the tab
  4. head is used to store the metadata about the document/website and links to external files
  5. Meta is used to give the meta data of the website like the description
  6. Body is the main part of the html that contains all the part of the website which has to be shown in the screen only
