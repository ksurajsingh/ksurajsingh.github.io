IIFE
Immediately Invocked Function Expression
This is the way of expressing functions so that they get executed ASAP : lol
formal: IIFE is defined and executed immediately after its creation.
It is typically used to create a new scope and
avoid polluting the global namespace.
  1. It is wrapped in parenthesis to cover it into an expression.
  2. It is followed by an () to execute it right away.
  3. Created a privileged scope - useful for encapsulation.
for ex: 
      
      async function sleep(){
         return new Promise((resolve,reject)=<{
         setTimeout(()=<{
           resolve(45)
         },1000)
         })
      }

      (function main(){
      let a = await sleep();
      let b = await sleep();
      })()
  • Since you are wrapping the function inside the parenthesis make sure to terminate the previous statement with a ;
